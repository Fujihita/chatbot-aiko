About:Aiko
a server-side chat bot by Fujihita

This is a chat logger bot written in Nodejs and hosted on Azure cloud for Kantai Collection English Wikia chat room. The bot accesses Wikia's Special:Chat controller via socket.io interface directly.

  • To stop the bot, visit /api/stop
  • To start the bot again, visit /api/start
  • To check the bot's status, visit /api/status

Currently, there is no database or external log and due to the nature of Azure free cloud hosting, all chat logs are not permanent. They'll be reset every 29 hours or 24 minutes without activities.

Perhaps, this is for the best...

Check out the latest documents at https://github.com/Fujihita/chatbot-aiko
